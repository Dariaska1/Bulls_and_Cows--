# Bulls_and_Cows на C++

## Описание
Классическая логическая игра "Быки и Коровы", написанная на C++

## Возможности
- Три режима игры:
  1. Против компьютера
  2. Игрок против игрока
  3. Два игрока против друг друга
- Настраиваемая длина числа (1-10 цифр)
- Возможность разрешить/запретить ноль в начале
- Ограничение по количеству попыток
- Сохранение логов игры в файл
